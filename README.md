# PT Putra Taro Paloma - Company Profile Website

Website company profile dan landing page perusahaan yang dirancang untuk menampilkan informasi perusahaan secara profesional, modern, dan mudah diakses oleh publik. Website ini membantu perusahaan dalam menyampaikan informasi profil, berita, layanan, serta kontak perusahaan secara digital.

---

## 📌 Tentang Project

Project ini merupakan website company profile untuk : meningkatkan branding perusahaan dan memudahkan masyarakat maupun calon mitra dalam mendapatkan informasi perusahaan secara online.

Website dibuat dengan tampilan modern, responsive, dan user friendly sehingga dapat diakses dengan baik melalui desktop maupun mobile device.

---

## 🚀 Cara Menjalankan Project

### 1. Clone Repository

```bash
git clone https://github.com/username/nama-repository.git
```

### 2. Masuk ke Folder Project

```bash
cd nama-repository
```

### 3. Install Dependency

```bash
composer install
```

### 4. Copy File Environment

```bash
cp .env.example .env
```

### 5. Generate Key Laravel

```bash
php artisan key:generate
```

### 6. Konfigurasi Database

Atur database pada file `.env`

```env
DB_DATABASE=nama_database
DB_USERNAME=root
DB_PASSWORD=
```

### 7. Jalankan Migration

```bash
php artisan migrate
```

### 8. Jalankan Server

```bash
php artisan serve
```
